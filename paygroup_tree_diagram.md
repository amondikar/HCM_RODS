## Key Branches Explained

### 🏢 Employment Branch (Most Complex)
The deepest branch with 12 levels, containing:
- Assignment details (job, position, grade)
- Manager hierarchy
- Department and location
- Salary costing
- Tax reporting units
- Custom fields

### 👤 Person Information Branches
Multiple parallel branches for different aspects:
- **Names**: GlobalPersonData, LocalPersonNameData
- **Contact**: EmailDataHistory, PhoneDataHistory, AddressDataHistory
- **Identity**: PersonIdentifiersDataHistory, PassportDataHistory, VisaAndPermitData
- **Demographics**: CitizenshipDataHistory, PersonDetailsDataHistory

### 💰 Compensation Branch
- **ElementData**: Contains all pay elements, allowances, deductions
- **SalaryCostingDataHistoryRT**: Accounting/GL allocation

### 👨‍👩‍👧‍👦 Relationships Branch
- **ContactData**: Emergency contacts, dependents, beneficiaries

### 📄 Documents & Custom Fields
- **CustomDocumentsData**: Bank details, custom documents
- **CustomPersonEITSGData**: Singapore-specific tax and CPF data
- **ThirdPartyIdentifiersData**: External system IDs
